# Mini-Projeto-M2
API RESTful desenvolvida em Node.js com Express e Sequelize para gerenciar tarefas. Permite criar, listar, buscar, atualizar e excluir tarefas, armazenadas em banco SQLite. Segue arquitetura MVC, valida dados de entrada e utiliza variáveis de ambiente para configuração, garantindo organização e escalabilidade.
